# 圣地巡礼地图总览 / Anime Pilgrimage Map Overview

<Callout type="note">这是站内的资源型入口页：适合作为外链落地页（比直接指向某篇正文更容易被引用）。</Callout>

这个页面的目标是：让外站/外链优先指向“可引用资源”，而不是散落的正文文章。

## 你能在这里获得什么

- 所有已收录点位的汇总清单
- 每条路线的入口（带一键在 Google Maps 打开）
- 便于收藏/分享的固定 URL

## 使用方法

1. 先从点位列表挑选你要去的区域
2. 点击“在 Google Maps 打开”，选择步行/公共交通
3. 回到站内阅读对应的路线文章获取机位与避坑信息
